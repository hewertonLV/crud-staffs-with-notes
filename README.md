# crud-staffs-with-notes

## Description

CRUD of colaborators, jobs and notes with php 7 + laravel v.7

## Requirements
Composer v.2.6.2
Laravel v.7
MYSQL
PHP v.7.4

## Install project

create file .env with the correct values to ENVs
``` composer install ``` 
``` npm install ``` 
``` composer require yoeunes/toastr ``` 
``` php artisan migrate ```
``` php artisan db:seed ```

## Running project

``` php artisan serve ```

## URL

http://127.0.0.1:8000/colaborador/index
